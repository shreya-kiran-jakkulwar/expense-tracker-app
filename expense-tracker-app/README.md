# BudgetBuddy 💰

A simple yet powerful React-based expense tracking application designed to help you track, analyze, and optimize your finances with a smooth and intuitive experience. Built especially for students and young professionals who want to master their money without the complexity.

## ✨ Features
- **Smart Analytics**: Daily, weekly, monthly, and yearly insights with beautiful charts.
- **Personalized Themes**: Choose your vibe with our dark mode and 4 vibrant pastel themes.
- **Interactive Calendar**: Hover over any date to see your spending at a glance.
- **Category-based Tracking**: Understand where your money goes with detailed categorical breakdowns.
- **PDF Reports**: Export your dashboard as a professional PDF report to share or save for records.
- **Local Storage**: All your data is saved securely in your browser so you don't lose anything.

## 🚀 Tech Stack
- Frontend Framework: React (v19)
- Styling: Tailwind CSS & Vanilla CSS
- Animations: Framer Motion
- Charts: Recharts
- PDF Generation: jsPDF & html2canvas

## 🛠️ How to run locally
1. Clone the repository
2. Run `npm install` to install all dependencies
3. Start the server with `npm start`
4. The app will open automatically at `http://localhost:3000`

## 👨‍💻 Development Journey
Built iteratively over several days, focusing on:
1. Solid folder structure & reusable components.
2. Robust state management via Context API and custom hooks.
3. Engaging UI variants with dark/light themes and sleek micro-interactions.
4. Rich data visualization through an integrated analytics dashboard.
5. Final polish, edge cases handling, and export functionality.
